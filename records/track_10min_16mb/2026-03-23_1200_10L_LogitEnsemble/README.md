# 10L + Logit Ensemble

## 목적
SWA 평균 모델과 마지막 학습 체크포인트의 logit을 평균하여 앙상블 효과 획득.
가중치 공간 평균(SWA)과 logit 공간 평균은 다른 정보를 제공할 수 있음.

## 변경 사항 (vs SOTA)
- 학습: 완전 동일
- 평가: SWA 모델 logits + 마지막 체크포인트 logits 평균

## 예상 결과
- val_bpb: 1.140~1.143 (앙상블로 -0.001~0.002 bpb)
- 평가 시간: ~2배 (2개 모델 forward)
